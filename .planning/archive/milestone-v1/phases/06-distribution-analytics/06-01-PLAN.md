    **`uploadVideoToYouTube(params: { accessToken: string; videoBuffer: Buffer; title: string; description: string; tags: string[]; categoryId?: string; privacyStatus: 'private' | 'unlisted' | 'public'; publishAt?: string; thumbnailBuffer?: Buffer; onProgress?: (percent: number) => void; }): Promise<{ youtubeVideoId: string; videoUrl: string; }>`**

    Implementation steps:

    1. Import `google` from `googleapis`. Create an OAuth2 client and set credentials with the user's access token (obtained from better-auth session — Google OAuth provides refresh tokens).
    2. Create a youtube client: `google.youtube({ version: "v3", auth: oauth2Client })`.
    3. Build the request body:
       ```typescript
       const requestBody = {
         snippet: {
           title: params.title,
           description: params.description,
           tags: params.tags,
           categoryId: params.categoryId || "22",
         },
         status: {
           privacyStatus: params.privacyStatus,
           selfDeclaredMadeForKids: false,
           ...(params.publishAt && { publishAt: params.publishAt }),
         },
       };
       ```
    4. Use `youtube.videos.insert` with resumable upload:
       ```typescript
       const res = await youtube.videos.insert({
         part: ["snippet", "status"],
         requestBody,
         media: {
           mimeType: "video/mp4",
           body: Readable.from(params.videoBuffer),
         },
       }, {
         onUploadProgress: (evt) => {
           if (params.onProgress && evt.bytesRead) {
             const percent = Math.round((evt.bytesRead / params.videoBuffer.length) * 100);
             params.onProgress(percent);
           }
         },
       });
       ```
    5. Extract the video ID from `res.data.id`.
    6. If `params.thumbnailBuffer` is provided, upload the thumbnail:
       ```typescript
       await youtube.thumbnails.set({
         videoId: res.data.id!,
         media: {
           mimeType: "image/png",
           body: Readable.from(params.thumbnailBuffer),
         },
       });
       ```
    7. Return `{ youtubeVideoId: res.data.id!, videoUrl: \`https://youtube.com/shorts/\${res.data.id}\` }`.

    **Key implementation details:**
    - Import `Readable` from `stream` to convert Buffer to ReadableStream for googleapis.
    - Import `google, Auth` from `googleapis`.
    - Create OAuth2Client: `new Auth.OAuth2Client()` then `oauth2Client.setCredentials({ access_token: params.accessToken })`.
    - If `publishAt` is set, `privacyStatus` MUST be `"private"` — add a validation check at the start of the function and throw if `publishAt` is set but privacyStatus is not "private".
    - Handle API errors: catch googleapis errors and throw a descriptive error with the YouTube API error message.
