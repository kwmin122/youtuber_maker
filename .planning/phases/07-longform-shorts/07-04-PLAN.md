# Plan 07-04: Clip Job + Child Project Generation

**Phase**: 7 — Long-form to Shorts Clipping
**Wave**: 3
**Dependencies**: 07-01, 07-02, 07-03
**Blast radius**: large (creates cross-table child projects, touches scenes/media_assets)
**Estimated files**: ~6 created, ~2 modified

## Goal

Implement the `longform-clip` job so that given a `longform_candidates.id` (or an array of them), the worker downloads the source once, FFmpeg-clips each selected candidate to a 1080x1920 30fps mp4, uploads to Supabase Storage, creates a child `projects` row, links it via `projects.parent_longform_id`, and seeds a synthetic `scripts`+`scenes`+`media_assets` chain so v1's editing/distribution pipeline can consume the clip unchanged.

## Scope

1. `clipLongform9x16` FFmpeg spawn helper
2. Disk preflight checker
3. Child project factory (DB transaction creating projects/scripts/scenes/media_assets rows)
4. `longform-clip` job handler (handles single or batch candidates, reuses downloaded source)
5. `POST /api/longform/candidates/clip` endpoint (accepts single or list of candidate IDs)
6. "Auto-clip all" convenience — endpoint + handler branch
7. Unit tests

### Task 1: FFmpeg 9:16 clip helper

**File**: `src/lib/video/clip-longform.ts` (NEW)

**Action**: spawn FFmpeg with input seek for speed:

```
ffmpeg -y -ss <startSec> -to <endSec> -i <input> \
  -vf "crop=ih*9/16:ih:(iw-ih*9/16)/2:0,scale=1080:1920,fps=30" \
  -c:v libx264 -preset veryfast -crf 23 \
  -c:a aac -b:a 128k -ar 44100 \
  -movflags +faststart \
  <output>
```

```ts
import { spawn } from "child_process";
import { parseFFmpegProgress } from "./ffmpeg-export";

export type ClipRequest = {
  inputPath: string;
  outputPath: string;
  startMs: number;
  endMs: number;
};

export async function clipLongform9x16(
  req: ClipRequest,
  onProgress?: (percent: number) => void
): Promise<void> {
  const startSec = (req.startMs / 1000).toFixed(3);
  const endSec = (req.endMs / 1000).toFixed(3);
  const duration = (req.endMs - req.startMs) / 1000;

  return new Promise((resolve, reject) => {
    const args = [
      "-y",
      "-ss", startSec,
      "-to", endSec,
      "-i", req.inputPath,
      "-vf", "crop=ih*9/16:ih:(iw-ih*9/16)/2:0,scale=1080:1920,fps=30",
      "-c:v", "libx264",
      "-preset", "veryfast",
      "-crf", "23",
      "-c:a", "aac",
      "-b:a", "128k",
      "-ar", "44100",
      "-movflags", "+faststart",
      req.outputPath,
    ];

    const proc = spawn("ffmpeg", args);
    let stderrTail = "";
    proc.stderr.on("data", (data) => {
      const chunk = data.toString();
      stderrTail = (stderrTail + chunk).slice(-4000);
      if (onProgress) {
        const lines = chunk.split(/\r|\n/);
        for (const line of lines) {
          const pct = parseFFmpegProgress(line, duration);
          if (pct !== null) onProgress(pct);
        }
      }
    });
    proc.on("close", (code) => {
      if (code === 0) resolve();
      else reject(new Error(`ffmpeg clip failed (${code}): ${stderrTail}`));
    });
    proc.on("error", (err) => reject(err));
  });
}
```

### Task 2: Disk preflight helper

**File**: `src/lib/video/disk-preflight.ts` (NEW)

**Action**: check free space under `os.tmpdir()` using `statfs` (Node 18+). Throws if free space < required.

```ts
import { statfs } from "fs/promises";
import { tmpdir } from "os";

export async function assertDiskSpaceAvailable(requiredBytes: number): Promise<void> {
  try {
    const stats = await statfs(tmpdir());
    const freeBytes = stats.bavail * stats.bsize;
    if (freeBytes < requiredBytes) {
      throw new Error(
        `Insufficient disk space in ${tmpdir()}: need ${requiredBytes} bytes, have ${freeBytes}`
      );
    }
  } catch (err) {
    // On platforms where statfs is unavailable, skip the preflight but log
    if ((err as NodeJS.ErrnoException).code === "ENOSYS") return;
    throw err;
  }
}
```

**IMPORTANT**: wrap the 5 inserts/updates in a single `db.transaction(async (tx) => { ... })` call so partial failures roll back cleanly.

```ts
import { db } from "@/lib/db";
import { projects, scripts, scenes, mediaAssets, longformCandidates, longformSources } from "@/lib/db/schema";
import { eq } from "drizzle-orm";
import { DEFAULT_SUBTITLE_STYLE } from "@/lib/video/types";

export async function createChildProjectForClip(params: {
  userId: string;
  source: typeof longformSources.$inferSelect;
  candidate: typeof longformCandidates.$inferSelect;
  clipPublicUrl: string;
  clipStoragePath: string;
}): Promise<{ projectId: string }> {
  const { userId, source, candidate, clipPublicUrl, clipStoragePath } = params;
  const durationSeconds = (candidate.endMs - candidate.startMs) / 1000;
  const title = candidate.titleSuggestion?.trim() || source.title || "Longform clip";

  return await db.transaction(async (tx) => {
    const [project] = await tx.insert(projects).values({
      userId,
      title,
      description: candidate.reason,
      parentLongformId: source.id,
      workflowState: {
        currentStep: 4,
        lastActiveTab: "video",
        completedSteps: [1, 2, 3],
        lastEditedAt: new Date().toISOString(),
        draftFlags: { fromLongform: true },
      },
    }).returning();

    const [script] = await tx.insert(scripts).values({
      projectId: project.id,
      analysisId: null, // requires Plan 07-01 to have made analysisId nullable; see Risks
      title,
      content: candidate.transcriptSnippet ?? "",
      variant: "longform",
      hookType: "longform",
      structureType: "longform-clip",
      wordCount: (candidate.transcriptSnippet ?? "").split(/\s+/).filter(Boolean).length,
      estimatedDuration: Math.round(durationSeconds),
      isSelected: true,
      aiProvider: "gemini",
    }).returning();

    const [scene] = await tx.insert(scenes).values({
      scriptId: script.id,
      sceneIndex: 0,
      narration: candidate.transcriptSnippet ?? "",
      imagePrompt: "",
      videoPrompt: "",
      duration: durationSeconds,
      subtitleStyle: DEFAULT_SUBTITLE_STYLE,
      transitionType: "cut",
      transitionDuration: 0,
      sourceType: "longform-clip",
      sourceClipStartMs: candidate.startMs,
      sourceClipEndMs: candidate.endMs,
      sourceLongformId: source.id,
    }).returning();

    await tx.insert(mediaAssets).values({
      sceneId: scene.id,
      type: "video",
      url: clipPublicUrl,
      storagePath: clipStoragePath,
      provider: "ffmpeg-longform-clip",
      status: "completed",
      metadata: { sourceId: source.id, candidateId: candidate.id },
    });

    await tx.update(longformCandidates).set({
      selected: true,
      childProjectId: project.id,
    }).where(eq(longformCandidates.id, candidate.id));

    return { projectId: project.id };
  });
}
```

### Task 5: Implement `longform-clip` handler

**File**: `src/worker/handlers/longform-clip.ts` (MODIFY — replace the stub)

**Action**: accept a batch payload:

```ts
payload: { sourceId: string; candidateIds: string[] }
```

Flow:
1. Load source, verify `status = 'analyzed'`, load all referenced candidates (verify they belong to the source)
2. `assertDiskSpaceAvailable((source.metadata.filesizeApprox ?? source.durationSeconds * 1_000_000) * 2)` — rough 2x heuristic
3. Create tempDir, download source MP4 once
4. For each candidate (loop, not parallel — FFmpeg is CPU-bound):
   - Update `jobs.currentStep = 'clipping N/M'`, `progress = 10 + Math.round(80 * i / candidates.length)`
   - `clipLongform9x16({ inputPath, outputPath, startMs, endMs })`
   - Read clipped file buffer
   - Call `uploadLongformClip` to push to Supabase `media` bucket — but we need a `childProjectId` first. Workaround: first create the child project with a placeholder media URL, then update. Simplest: assign a fresh UUID `crypto.randomUUID()` and use it for the storage path BEFORE creating the project, then pass it to `createChildProjectForClip` as `predeterminedProjectId` — but `projects.id` is `defaultRandom()` and we cannot pre-assign easily.
   - **Cleaner approach**: do a two-phase insert. Phase A: upload clipped buffer to a temp path `longform-clips/pending/${candidateId}.mp4`. Phase B: call `createChildProjectForClip` which returns a new project id. Phase C: copy/move the storage object to the final path `${userId}/${projectId}/longform-clips/clip-${candidateId}.mp4`, then update `media_assets.url` and `media_assets.storagePath`.
   - **Simpler MVP approach**: just upload under a stable path `${userId}/longform-clips/${candidateId}.mp4` (no projectId in path). Accept that the `media` bucket path convention is not strictly followed for longform clips. This is acceptable because the `media_assets.storagePath` is stored in DB and deletion works correctly by path, not by convention.
   - Use the simpler approach: upload to `${userId}/longform-clips/${candidateId}.mp4` directly via `supabase.storage.from('media').upload(path, buffer)` and get the public URL. Add a new helper in `src/lib/media/longform-storage.ts` named `uploadLongformClipBuffer` that doesn't require a projectId.

```ts
export async function uploadLongformClipBuffer(params: {
  userId: string;
  candidateId: string;
  buffer: Buffer;
}): Promise<{ storagePath: string; publicUrl: string }> {
  const { createSupabaseClient } = await import("@/lib/supabase");
  const supabase = createSupabaseClient();
  const storagePath = `${params.userId}/longform-clips/${params.candidateId}.mp4`;
  const { error } = await supabase.storage.from("media").upload(storagePath, params.buffer, {
    contentType: "video/mp4",
    upsert: true,
  });
  if (error) throw new Error(`longform clip upload failed: ${error.message}`);
  const { data } = supabase.storage.from("media").getPublicUrl(storagePath);
  return { storagePath, publicUrl: data.publicUrl };
}
```

```ts
import type { Job } from "bullmq";
import { eq, inArray, and } from "drizzle-orm";
import { mkdtemp, rm, writeFile, readFile } from "fs/promises";
import { join } from "path";
import { tmpdir } from "os";
import { jobs, jobEvents, longformSources, longformCandidates } from "@/lib/db/schema";
import { downloadLongformSource, uploadLongformClipBuffer } from "@/lib/media/longform-storage";
import { clipLongform9x16 } from "@/lib/video/clip-longform";
import { assertDiskSpaceAvailable } from "@/lib/video/disk-preflight";
import { createChildProjectForClip } from "@/lib/longform/create-child-project";

// ... same DrizzleInstance type
export async function handleLongformClip(job: Job, db: DrizzleInstance) {
  const { jobId, userId, payload } = job.data as {
    jobId: string; userId: string;
    payload: { sourceId: string; candidateIds: string[] };
  };
  const { sourceId, candidateIds } = payload;
  if (!candidateIds?.length) throw new Error("candidateIds required and non-empty");

  let tempDir: string | null = null;
  try {
    await db.update(jobs).set({ status: "active", currentStep: "loading source", updatedAt: new Date() }).where(eq(jobs.id, jobId));
    await db.insert(jobEvents).values({ jobId, event: "started", data: { sourceId, count: candidateIds.length } });

    const [source] = await db.select().from(longformSources).where(eq(longformSources.id, sourceId));
    if (!source) throw new Error(`source ${sourceId} not found`);
    if (source.status !== "analyzed") throw new Error(`source not analyzed: ${source.status}`);
    if (!source.storagePath) throw new Error("source has no storagePath");

    const candidates = await db.select().from(longformCandidates)
      .where(and(eq(longformCandidates.sourceId, sourceId), inArray(longformCandidates.id, candidateIds)));
    if (candidates.length !== candidateIds.length) {
      throw new Error(`expected ${candidateIds.length} candidates, found ${candidates.length}`);
    }

    // Disk preflight — estimate source size ~= durationSec * ~500 KB/s * 2x working copy
    const estimated = (source.durationSeconds ?? 600) * 500_000 * 2;
    await assertDiskSpaceAvailable(estimated);

    tempDir = await mkdtemp(join(tmpdir(), "longform-clip-"));
    const sourcePath = join(tempDir, "source.mp4");

    await db.update(jobs).set({ currentStep: "downloading source", progress: 5, updatedAt: new Date() }).where(eq(jobs.id, jobId));
    const sourceBuffer = await downloadLongformSource(source.storagePath);
    await writeFile(sourcePath, sourceBuffer);

    const childProjectIds: string[] = [];

    for (let i = 0; i < candidates.length; i++) {
      const cand = candidates[i];
      const outputPath = join(tempDir, `clip-${cand.id}.mp4`);
      const progressBase = 10 + Math.round(80 * i / candidates.length);

      await db.update(jobs).set({
        currentStep: `clipping ${i + 1}/${candidates.length}`,
        progress: progressBase,
        updatedAt: new Date(),
      }).where(eq(jobs.id, jobId));

      await clipLongform9x16({
        inputPath: sourcePath,
        outputPath,
        startMs: cand.startMs,
        endMs: cand.endMs,
      });

      const clipBuffer = await readFile(outputPath);
      const { storagePath: clipStoragePath, publicUrl: clipPublicUrl } = await uploadLongformClipBuffer({
        userId,
        candidateId: cand.id,
        buffer: clipBuffer,
      });

      const { projectId } = await createChildProjectForClip({
        userId,
        source,
        candidate: cand,
        clipPublicUrl,
        clipStoragePath,
      });
      childProjectIds.push(projectId);

      await db.insert(jobEvents).values({
        jobId,
        event: "progress",
        data: { candidateId: cand.id, projectId, done: i + 1, total: candidates.length },
      });

      await rm(outputPath, { force: true });
    }

    await db.update(jobs).set({
      status: "completed",
      progress: 100,
      currentStep: "clipping complete",
      result: { sourceId, childProjectIds, count: childProjectIds.length },
      updatedAt: new Date(),
    }).where(eq(jobs.id, jobId));
    await db.insert(jobEvents).values({ jobId, event: "completed", data: { childProjectIds } });

    return { childProjectIds };
  } catch (error) {
    const errorMessage = error instanceof Error ? error.message : String(error);
    await db.update(jobs).set({ status: "failed", errorMessage, updatedAt: new Date() }).where(eq(jobs.id, jobId));
    await db.insert(jobEvents).values({ jobId, event: "failed", data: { error: errorMessage } });
    throw error;
  } finally {
    if (tempDir) await rm(tempDir, { recursive: true, force: true });
  }
}
```

### Task 6: API route for clipping

**File**: `src/app/api/longform/candidates/clip/route.ts` (NEW)

**Action**: body:
```ts
{ sourceId: string; candidateIds: string[] }
```
or
```ts
{ sourceId: string; all: true } // auto-clip all
```

Validates that the source belongs to the user, resolves "all" by loading every candidate id for the source, then enqueues one `longform-clip` job.

```ts
import { NextRequest, NextResponse } from "next/server";
import { z } from "zod";
import { and, eq } from "drizzle-orm";
import { db } from "@/lib/db";
import { jobs, longformCandidates, longformSources } from "@/lib/db/schema";
import { getLongformQueue } from "@/lib/queue-longform";
import { getServerSession } from "@/lib/auth/get-session";

const bodySchema = z.discriminatedUnion("mode", [
  z.object({ mode: z.literal("selected"), sourceId: z.string().uuid(), candidateIds: z.array(z.string().uuid()).min(1) }),
  z.object({ mode: z.literal("all"), sourceId: z.string().uuid() }),
]);

export async function POST(request: NextRequest) {
  const session = await getServerSession();
  if (!session?.user) return NextResponse.json({ error: "Unauthorized" }, { status: 401 });
  const parsed = bodySchema.safeParse(await request.json().catch(() => null));
  if (!parsed.success) return NextResponse.json({ error: "Invalid body", details: parsed.error.flatten() }, { status: 400 });

  const body = parsed.data;

  const [source] = await db.select().from(longformSources).where(and(eq(longformSources.id, body.sourceId), eq(longformSources.userId, session.user.id)));
  if (!source) return NextResponse.json({ error: "Not found" }, { status: 404 });
  if (source.status !== "analyzed") return NextResponse.json({ error: `source not analyzed (${source.status})` }, { status: 409 });

  let candidateIds: string[];
  if (body.mode === "all") {
    const rows = await db.select({ id: longformCandidates.id }).from(longformCandidates).where(eq(longformCandidates.sourceId, body.sourceId));
    candidateIds = rows.map((r) => r.id);
  } else {
    candidateIds = body.candidateIds;
  }
  if (candidateIds.length === 0) return NextResponse.json({ error: "no candidates" }, { status: 400 });

  const [jobRow] = await db.insert(jobs).values({
    userId: session.user.id,
    type: "longform-clip",
    status: "pending",
    progress: 0,
    payload: { sourceId: body.sourceId, candidateIds },
  }).returning();

  await getLongformQueue().add("longform-clip", {
    payload: { sourceId: body.sourceId, candidateIds },
    jobId: jobRow.id,
    userId: session.user.id,
  });

  return NextResponse.json({ jobId: jobRow.id, count: candidateIds.length }, { status: 201 });
}
```

## Risks & Mitigations

- **Risk**: Transaction isolation may fail if `db` instance doesn't support `.transaction`. **Mitigation**: Drizzle + postgres-js does support `db.transaction(async tx => ...)`. Verified via existing codebase references.
- **Risk**: 10 clips × 500 MB intermediate buffers could blow memory. **Mitigation**: per-iteration `rm(outputPath)` and the clips are small (~10-30 MB each for 60s 1080p60 mp4 at crf 23).
- **Risk**: Two-phase upload path (upload before project exists) creates orphan storage objects on failure. **Mitigation**: flat `${userId}/longform-clips/${candidateId}.mp4` path is stable; orphan cleanup can be a background job later. Not a blocker for MVP.
- **Risk**: FFmpeg input seek (`-ss` before `-i`) is fast but not frame-accurate. **Mitigation**: acceptable for shorts (±50ms not noticeable). Can switch to output seek (`-ss` after `-i`) later if precision matters.
