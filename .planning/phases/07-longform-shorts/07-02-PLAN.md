## Goal

Implement the `longform-download` job so that a user can submit either a YouTube URL or a direct uploaded file, and end up with a row in `longform_sources` with `status='ready'`, a Supabase Storage path, and extracted metadata (duration, title). This plan also ships the API surface the UI (Plan 07-05) will call.

## Scope

1. `yt-dlp` spawn wrapper `src/lib/video/ytdlp.ts`
2. `longform-download` job handler — URL mode (yt-dlp) and file mode (copy from Supabase)
3. `POST /api/longform/sources` — creates a `longform_sources` row and enqueues `longform-download`
4. `POST /api/longform/sources/upload-url` — returns a signed upload URL for direct Supabase resumable upload
5. `GET /api/longform/sources/[id]` — fetches a source with its candidates
6. Duration bounds enforcement (120-14400 seconds)
7. Unit tests for ytdlp wrapper + handler

### Task 1: `yt-dlp` wrapper

**File**: `src/lib/video/ytdlp.ts` (NEW)

**Action**: create a thin spawn-based wrapper that exposes two functions:

1. `probeVideoMetadata(url: string)` — runs `yt-dlp --dump-json --no-warnings --skip-download <url>` and parses stdout JSON for `{ duration, title, id, webpage_url, ext, filesize_approx, format_id }`.
2. `downloadVideo(url: string, outputPath: string, onProgress?: (pct: number) => void)` — runs `yt-dlp -f "bv*[height<=720]+ba/b[height<=720]" --merge-output-format mp4 -o <outputPath> <url>` and parses stderr lines for progress.

Use `child_process.spawn` — NEVER `exec` or `execSync`. Mirror the error handling of `src/lib/video/ffmpeg-export.ts`.

```ts
import { spawn } from "child_process";

export type YtDlpMetadata = {
  id: string;
  title: string;
  durationSeconds: number;
  webpageUrl: string;
  ext: string;
  filesizeApprox: number | null;
};

export async function probeVideoMetadata(url: string): Promise<YtDlpMetadata> {
  return new Promise((resolve, reject) => {
    const args = ["--dump-json", "--no-warnings", "--skip-download", url];
    const proc = spawn("yt-dlp", args);

    let stdout = "";
    let stderr = "";
    proc.stdout.on("data", (d) => (stdout += d.toString()));
    proc.stderr.on("data", (d) => (stderr += d.toString()));

    proc.on("close", (code) => {
      if (code !== 0) {
        return reject(new Error(`yt-dlp probe failed (${code}): ${stderr}`));
      }
      try {
        const json = JSON.parse(stdout);
        resolve({
          id: json.id,
          title: json.title,
          durationSeconds: Math.round(json.duration),
          webpageUrl: json.webpage_url,
          ext: json.ext,
          filesizeApprox: json.filesize_approx ?? null,
        });
      } catch (err) {
        reject(new Error(`yt-dlp probe JSON parse failed: ${(err as Error).message}`));
      }
    });

    proc.on("error", (err) => reject(new Error(`yt-dlp spawn failed: ${err.message}`)));
  });
}

export async function downloadVideo(
  url: string,
  outputPath: string,
  onProgress?: (percent: number) => void
): Promise<void> {
  return new Promise((resolve, reject) => {
    const args = [
      "-f", "bv*[height<=720]+ba/b[height<=720]",
      "--merge-output-format", "mp4",
      "--no-warnings",
      "-o", outputPath,
      url,
    ];
    const proc = spawn("yt-dlp", args);

    let stderrTail = "";
    proc.stdout.on("data", (d) => {
      const line = d.toString();
      const match = line.match(/\[download\]\s+(\d+\.\d+)%/);
      if (match && onProgress) {
        onProgress(parseFloat(match[1]));
      }
    });
    proc.stderr.on("data", (d) => {
      stderrTail = (stderrTail + d.toString()).slice(-4000);
    });

    proc.on("close", (code) => {
      if (code === 0) resolve();
      else reject(new Error(`yt-dlp download failed (${code}): ${stderrTail}`));
    });
    proc.on("error", (err) => reject(new Error(`yt-dlp spawn failed: ${err.message}`)));
  });
}
```

### Task 3: Implement `longform-download` handler

**File**: `src/worker/handlers/longform-download.ts` (MODIFY — replace the stub from Plan 07-01)

**Action**: implement the full handler. Two code paths depending on `longform_sources.sourceType`:

**URL path**:
1. Set `jobs.status='active'`, `currentStep='probing video'`, insert `job_events` "started"
2. `probeVideoMetadata(source.sourceUrl)` → extract duration, title
3. `assertDurationInBounds(duration)` — fail early
4. Create `tempDir` via `mkdtemp(join(tmpdir(), 'longform-dl-'))`
5. `downloadVideo(url, join(tempDir, 'source.mp4'), onProgress)` — forward progress to `jobs.progress` + `job_events`
6. Read the downloaded file buffer and call `uploadLongformSource` (Task 5) to push to Supabase bucket `longform-sources` under path `${userId}/${sourceId}/source.mp4`
7. Update `longform_sources` row: `status='ready'`, `storagePath`, `publicUrl`, `title`, `durationSeconds`, `metadata={ ytdlpId, webpageUrl }`
8. `jobs.status='completed'`, `progress=100`, `result={ sourceId }`
9. `finally`: `rm(tempDir, { recursive: true, force: true })`

**File path**:
1. Source row already has `storagePath` (the direct-upload path the client wrote to Supabase)
2. Download it via `supabase.storage.from('longform-sources').download(storagePath)` to a tempDir
3. Probe duration via `ffprobe` — use `spawn('ffprobe', ['-v','error','-show_entries','format=duration:format_tags=title','-of','json', tempPath])`. Parse JSON, extract `format.duration` (seconds, float).
4. `assertDurationInBounds(duration)` — if fails, delete the uploaded file and update status='failed'
5. Update `longform_sources` row with `durationSeconds`, `title` (fallback to filename), `status='ready'`, `publicUrl` from Supabase
6. Same cleanup/finally pattern

Structure to mirror `src/worker/handlers/export-video.ts` — same try/catch/finally/jobEvents pattern. The handler MUST read `source_id` from `job.data.payload.sourceId`.

```ts
import type { Job } from "bullmq";
import { eq } from "drizzle-orm";
import { mkdtemp, rm, writeFile, readFile, stat } from "fs/promises";
import { join } from "path";
import { tmpdir } from "os";
import { spawn } from "child_process";
import { jobs, jobEvents, longformSources } from "@/lib/db/schema";
import { probeVideoMetadata, downloadVideo } from "@/lib/video/ytdlp";
import { assertDurationInBounds, LONGFORM_BUCKET } from "@/lib/video/longform-constants";
import { uploadLongformSource, downloadLongformSource } from "@/lib/media/longform-storage";

type DrizzleInstance = {
  update: typeof import("@/lib/db").db.update;
  insert: typeof import("@/lib/db").db.insert;
  select: typeof import("@/lib/db").db.select;
  delete: typeof import("@/lib/db").db.delete;
};

export async function handleLongformDownload(job: Job, db: DrizzleInstance) {
  const { jobId, userId, payload } = job.data as {
    jobId: string;
    userId: string;
    payload: { sourceId: string };
  };
  const { sourceId } = payload;

  let tempDir: string | null = null;
  try {
    await db.update(jobs).set({ status: "active", currentStep: "loading source", updatedAt: new Date() }).where(eq(jobs.id, jobId));
    await db.insert(jobEvents).values({ jobId, event: "started", data: { sourceId } });

    const [source] = await db.select().from(longformSources).where(eq(longformSources.id, sourceId));
    if (!source) throw new Error(`longform_source ${sourceId} not found`);

    await db.update(longformSources).set({ status: "downloading", updatedAt: new Date() }).where(eq(longformSources.id, sourceId));

    tempDir = await mkdtemp(join(tmpdir(), "longform-dl-"));
    let finalPath: string;
    let durationSeconds: number;
    let title: string;
    let metadata: Record<string, unknown> = {};

    if (source.sourceType === "url") {
      if (!source.sourceUrl) throw new Error("URL source missing sourceUrl");

      await db.update(jobs).set({ currentStep: "probing video", progress: 5, updatedAt: new Date() }).where(eq(jobs.id, jobId));
      const meta = await probeVideoMetadata(source.sourceUrl);
      assertDurationInBounds(meta.durationSeconds);
      title = meta.title;
      durationSeconds = meta.durationSeconds;
      metadata = { ytdlpId: meta.id, webpageUrl: meta.webpageUrl };

      finalPath = join(tempDir, "source.mp4");
      await db.update(jobs).set({ currentStep: "downloading video", progress: 10, updatedAt: new Date() }).where(eq(jobs.id, jobId));

      await downloadVideo(source.sourceUrl, finalPath, async (pct) => {
        const scaled = 10 + Math.round(pct * 0.6); // 10-70% range for download
        await db.update(jobs).set({ progress: scaled, updatedAt: new Date() }).where(eq(jobs.id, jobId));
        await db.insert(jobEvents).values({ jobId, event: "progress", data: { phase: "download", percent: pct } });
      });
    } else if (source.sourceType === "file") {
      if (!source.storagePath) throw new Error("file source missing storagePath");
      finalPath = join(tempDir, "source.mp4");

      await db.update(jobs).set({ currentStep: "downloading uploaded file", progress: 10, updatedAt: new Date() }).where(eq(jobs.id, jobId));
      const buffer = await downloadLongformSource(source.storagePath);
      await writeFile(finalPath, buffer);

      const probe = await probeDurationWithFfprobe(finalPath);
      assertDurationInBounds(probe.duration);
      durationSeconds = probe.duration;
      title = probe.title ?? source.title ?? "Uploaded video";
      metadata = { probedViaFfprobe: true };
    } else {
      throw new Error(`Unknown sourceType: ${source.sourceType}`);
    }

    // Upload normalized source to Supabase (URL mode only; file mode already there)
    await db.update(jobs).set({ currentStep: "uploading to storage", progress: 75, updatedAt: new Date() }).where(eq(jobs.id, jobId));

    let storagePath = source.storagePath;
    let publicUrl = source.publicUrl;
    if (source.sourceType === "url") {
      const buffer = await readFile(finalPath);
      const uploaded = await uploadLongformSource({ userId, sourceId, buffer });
      storagePath = uploaded.storagePath;
      publicUrl = uploaded.publicUrl;
    } else {
      publicUrl = publicUrl ?? (await resolveLongformPublicUrl(source.storagePath!));
    }

    await db.update(longformSources).set({
      status: "ready",
      storagePath,
      publicUrl,
      title,
      durationSeconds,
      metadata,
      updatedAt: new Date(),
    }).where(eq(longformSources.id, sourceId));

    await db.update(jobs).set({
      status: "completed",
      progress: 100,
      currentStep: "download complete",
      result: { sourceId, durationSeconds, title },
      updatedAt: new Date(),
    }).where(eq(jobs.id, jobId));
    await db.insert(jobEvents).values({ jobId, event: "completed", data: { sourceId } });

    return { sourceId };
  } catch (error) {
    const errorMessage = error instanceof Error ? error.message : String(error);
    await db.update(jobs).set({ status: "failed", errorMessage, updatedAt: new Date() }).where(eq(jobs.id, jobId));
    await db.update(longformSources).set({ status: "failed", errorMessage, updatedAt: new Date() }).where(eq(longformSources.id, (job.data as { payload: { sourceId: string } }).payload.sourceId));
    await db.insert(jobEvents).values({ jobId, event: "failed", data: { error: errorMessage } });
    throw error;
  } finally {
    if (tempDir) await rm(tempDir, { recursive: true, force: true });
  }
}

async function probeDurationWithFfprobe(filePath: string): Promise<{ duration: number; title?: string }> {
  return new Promise((resolve, reject) => {
    const proc = spawn("ffprobe", [
      "-v", "error",
      "-show_entries", "format=duration:format_tags=title",
      "-of", "json",
      filePath,
    ]);
    let stdout = "";
    let stderr = "";
    proc.stdout.on("data", (d) => (stdout += d.toString()));
    proc.stderr.on("data", (d) => (stderr += d.toString()));
    proc.on("close", (code) => {
      if (code !== 0) return reject(new Error(`ffprobe failed: ${stderr}`));
      try {
        const json = JSON.parse(stdout);
        resolve({
          duration: Math.round(parseFloat(json.format?.duration ?? "0")),
          title: json.format?.tags?.title,
        });
      } catch (err) {
        reject(new Error(`ffprobe JSON parse failed: ${(err as Error).message}`));
      }
    });
    proc.on("error", (err) => reject(err));
  });
}

async function resolveLongformPublicUrl(storagePath: string): Promise<string> {
  const { createSupabaseClient } = await import("@/lib/supabase");
  const supabase = createSupabaseClient();
  const { data } = supabase.storage.from(LONGFORM_BUCKET).getPublicUrl(storagePath);
  return data.publicUrl;
}
```

### Task 4: Longform storage helper

**File**: `src/lib/media/longform-storage.ts` (NEW)

**Action**: helpers for uploading/downloading longform sources. Mirrors `src/lib/media/storage.ts` but on the `longform-sources` bucket.

```ts
import { createSupabaseClient } from "@/lib/supabase";
import { LONGFORM_BUCKET } from "@/lib/video/longform-constants";

export async function uploadLongformSource(params: {
  userId: string;
  sourceId: string;
  buffer: Buffer;
}): Promise<{ storagePath: string; publicUrl: string }> {
  const supabase = createSupabaseClient();
  const storagePath = `${params.userId}/${params.sourceId}/source.mp4`;

  const { error } = await supabase.storage
    .from(LONGFORM_BUCKET)
    .upload(storagePath, params.buffer, {
      contentType: "video/mp4",
      upsert: true,
    });
  if (error) throw new Error(`Longform upload failed: ${error.message}`);

  const { data } = supabase.storage.from(LONGFORM_BUCKET).getPublicUrl(storagePath);
  return { storagePath, publicUrl: data.publicUrl };
}

export async function downloadLongformSource(storagePath: string): Promise<Buffer> {
  const supabase = createSupabaseClient();
  const { data, error } = await supabase.storage.from(LONGFORM_BUCKET).download(storagePath);
  if (error) throw new Error(`Longform download failed: ${error.message}`);
  const arrayBuffer = await data.arrayBuffer();
  return Buffer.from(arrayBuffer);
}

export async function deleteLongformSource(storagePath: string): Promise<void> {
  const supabase = createSupabaseClient();
  const { error } = await supabase.storage.from(LONGFORM_BUCKET).remove([storagePath]);
  if (error) throw new Error(`Longform delete failed: ${error.message}`);
}
```

### Task 5: `POST /api/longform/sources` — create source + enqueue job

**File**: `src/app/api/longform/sources/route.ts` (NEW)

**Action**: request body is a discriminated union:
- `{ sourceType: "url", url: string }` — validate with `parseVideoUrl`
- `{ sourceType: "file", storagePath: string, title?: string }` — validate that storagePath starts with `${userId}/`

On success:
1. Insert `longform_sources` row (`status: 'pending'`)
2. Insert `jobs` row (`type: 'longform-download'`)
3. Enqueue on `getLongformQueue()`
4. Return `{ sourceId, jobId }`

Mirror auth pattern from `src/app/api/jobs/route.ts` — use `getServerSession()`.

```ts
import { NextRequest, NextResponse } from "next/server";
import { z } from "zod";
import { db } from "@/lib/db";
import { jobs, longformSources } from "@/lib/db/schema";
import { getLongformQueue } from "@/lib/queue-longform";
import { getServerSession } from "@/lib/auth/get-session";
import { parseVideoUrl } from "@/lib/youtube/parse-url";

const bodySchema = z.discriminatedUnion("sourceType", [
  z.object({ sourceType: z.literal("url"), url: z.string().url() }),
  z.object({
    sourceType: z.literal("file"),
    storagePath: z.string().min(1),
    title: z.string().optional(),
  }),
]);

export async function POST(request: NextRequest) {
  const session = await getServerSession();
  if (!session?.user) return NextResponse.json({ error: "Unauthorized" }, { status: 401 });

  let body: unknown;
  try { body = await request.json(); } catch { return NextResponse.json({ error: "Invalid JSON" }, { status: 400 }); }
  const parsed = bodySchema.safeParse(body);
  if (!parsed.success) return NextResponse.json({ error: "Invalid body", details: parsed.error.flatten() }, { status: 400 });

  const data = parsed.data;

  if (data.sourceType === "url") {
    const parsedVideo = parseVideoUrl(data.url);
    if (!parsedVideo) return NextResponse.json({ error: "Not a valid YouTube video URL" }, { status: 400 });
  } else {
    if (!data.storagePath.startsWith(`${session.user.id}/`)) {
      return NextResponse.json({ error: "storagePath must be under your user folder" }, { status: 403 });
    }
  }

  const [source] = await db.insert(longformSources).values({
    userId: session.user.id,
    sourceType: data.sourceType,
    sourceUrl: data.sourceType === "url" ? data.url : null,
    storagePath: data.sourceType === "file" ? data.storagePath : null,
    title: data.sourceType === "file" ? (data.title ?? null) : null,
    status: "pending",
  }).returning();

  const [jobRow] = await db.insert(jobs).values({
    userId: session.user.id,
    type: "longform-download",
    status: "pending",
    progress: 0,
    payload: { sourceId: source.id },
  }).returning();

  await getLongformQueue().add("longform-download", {
    payload: { sourceId: source.id },
    jobId: jobRow.id,
    userId: session.user.id,
  });

  return NextResponse.json({ sourceId: source.id, jobId: jobRow.id }, { status: 201 });
}

export async function GET(_request: NextRequest) {
  const session = await getServerSession();
  if (!session?.user) return NextResponse.json({ error: "Unauthorized" }, { status: 401 });
  const rows = await db
    .select()
    .from(longformSources)
    .where((t, { eq: eqFn }) => eqFn(t.userId, session.user.id));
  return NextResponse.json(rows);
}
```

### Task 6: `POST /api/longform/sources/upload-url` — signed upload URL

**File**: `src/app/api/longform/sources/upload-url/route.ts` (NEW)

**Action**: body `{ filename: string, contentType: string, fileSize: number }`. Validate against `LONGFORM_ALLOWED_MIME_TYPES` and `LONGFORM_MAX_FILE_BYTES`. Call Supabase `createSignedUploadUrl`:

```ts
import { NextRequest, NextResponse } from "next/server";
import { z } from "zod";
import { randomUUID } from "crypto";
import { createSupabaseClient } from "@/lib/supabase";
import { getServerSession } from "@/lib/auth/get-session";
import {
  LONGFORM_ALLOWED_MIME_TYPES,
  LONGFORM_BUCKET,
  LONGFORM_MAX_FILE_BYTES,
} from "@/lib/video/longform-constants";

const bodySchema = z.object({
  filename: z.string().min(1).max(255),
  contentType: z.enum(LONGFORM_ALLOWED_MIME_TYPES),
  fileSize: z.number().int().positive().max(LONGFORM_MAX_FILE_BYTES),
});

export async function POST(request: NextRequest) {
  const session = await getServerSession();
  if (!session?.user) return NextResponse.json({ error: "Unauthorized" }, { status: 401 });
  const body = bodySchema.safeParse(await request.json().catch(() => null));
  if (!body.success) return NextResponse.json({ error: "Invalid body", details: body.error.flatten() }, { status: 400 });

  const uploadId = randomUUID();
  const storagePath = `${session.user.id}/uploads/${uploadId}/${body.data.filename}`;

  const supabase = createSupabaseClient();
  const { data, error } = await supabase.storage
    .from(LONGFORM_BUCKET)
    .createSignedUploadUrl(storagePath);

  if (error || !data) return NextResponse.json({ error: error?.message ?? "failed to sign" }, { status: 500 });

  return NextResponse.json({
    storagePath,
    token: data.token,
    signedUrl: data.signedUrl,
  });
}
```

```ts
import { NextRequest, NextResponse } from "next/server";
import { and, eq, asc } from "drizzle-orm";
import { db } from "@/lib/db";
import { longformSources, longformCandidates } from "@/lib/db/schema";
import { getServerSession } from "@/lib/auth/get-session";

export async function GET(
  _request: NextRequest,
  { params }: { params: Promise<{ id: string }> }
) {
  const session = await getServerSession();
  if (!session?.user) return NextResponse.json({ error: "Unauthorized" }, { status: 401 });
  const { id } = await params;

  const [source] = await db
    .select()
    .from(longformSources)
    .where(and(eq(longformSources.id, id), eq(longformSources.userId, session.user.id)));
  if (!source) return NextResponse.json({ error: "Not found" }, { status: 404 });

  const candidates = await db
    .select()
    .from(longformCandidates)
    .where(eq(longformCandidates.sourceId, id))
    .orderBy(asc(longformCandidates.startMs));

  return NextResponse.json({ source, candidates });
}
```

## Risks & Mitigations

- **Risk**: yt-dlp rate-limited from Railway IPs. **Mitigation**: retry with exponential backoff handled by BullMQ `attempts: 2, backoff: 5s`. Future: inject cookies via env var.
- **Risk**: 2 GB file decode into `Buffer` is memory-heavy. **Mitigation**: URL mode only uploads files it downloaded to disk, reading in one shot is acceptable up to ~1 GB. For larger files, switch to `createReadStream` + chunked upload (tracked as a follow-up, noted here).
- **Risk**: ffprobe binary unavailable. **Mitigation**: Dockerfile (07-01 Task 11) installs `ffmpeg` which ships ffprobe.
