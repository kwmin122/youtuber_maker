  <action>
    Create `src/app/api/music/search/route.ts` as a Next.js App Router route handler.

    **Query parameters accepted:**
    - `q` (string, required if genre not provided â€” at least one of q or genre must be present)
    - `genre` (string, optional) â€” maps to Pixabay `music_genre` param
    - `bpm` (string, optional) â€” ignored by Pixabay but accepted for future use / UI filtering

    **Normalized response type** (exported interface `PixabayTrack`):
    ```ts
    export interface PixabayTrack {
      id: number;
      title: string;    // from Pixabay `tags` field
      artist: string;   // from Pixabay `user` field
      url: string;      // from Pixabay `audio` field (CDN stream URL)
      previewUrl: string; // same as url â€” direct stream
      duration: number; // seconds, from Pixabay `duration`
    }
    ```

    **Implementation:**
    1. Call `getServerSession()` from `@/lib/auth/get-session`. Return 401 if no session.
    2. Read `q` and `genre` from `request.nextUrl.searchParams`.
    3. If both are absent or empty strings, return 400: `{ error: "query parameter 'q' or 'genre' is required" }`.
    4. Build Pixabay URL:
       ```
       const params = new URLSearchParams({ key: env.PIXABAY_API_KEY, per_page: "20" });
       if (q) params.set("q", q);
       if (genre) params.set("music_genre", genre);
       const pixabayUrl = `https://pixabay.com/api/music/?${params.toString()}`;
       ```
    5. `fetch(pixabayUrl)` with a 10-second AbortSignal timeout (`AbortSignal.timeout(10_000)`).
    6. If the upstream response is not ok, return 502: `{ error: "Pixabay API error", status: response.status }`.
    7. Parse JSON, map `hits` array to `PixabayTrack[]`:
       ```ts
       const tracks: PixabayTrack[] = data.hits.map((h: PixabayHit) => ({
         id: h.id,
         title: h.tags,
         artist: h.user,
         url: h.audio,
         previewUrl: h.audio,
         duration: h.duration,
       }));
       ```
    8. Return 200: `{ tracks }`.

    Define the internal `PixabayHit` interface (not exported):
    ```ts
    interface PixabayHit {
      id: number;
      tags: string;
      audio: string;
      duration: number;
      user: string;
    }
    ```

    Import `env` from `@/lib/env`, `getServerSession` from `@/lib/auth/get-session`, `NextRequest` and `NextResponse` from `next/server`.
  </action>
