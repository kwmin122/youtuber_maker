<objective>
This plan implements the OAuth 2.0 authorization flows for TikTok and Instagram so users can connect their accounts inside the app. TikTok uses the authorization code flow (D-03) with refresh-token rotation (D-06). Instagram uses Facebook Login OAuth with a long-lived token exchange (D-06). Both access tokens are stored in the existing Better Auth `account` table using `providerId: "tiktok"` / `"instagram"` (D-05). A new "연결된 계정" tab is added to `/settings` (D-04) showing connection status and Connect/Disconnect buttons for all three platforms (Google, TikTok, Instagram). When env vars are absent the TikTok/Instagram buttons render disabled (D-02).

Requirements fulfilled: MULTI-01
Phase goal contribution: Establishes the authenticated identity layer that the upload workers in Plan 02 rely on to retrieve valid access tokens before calling TikTok/Instagram APIs.
Complexity: M
</objective>

Export function `buildTikTokAuthUrl(state: string): string` that constructs the authorization URL with params: `client_key = process.env.TIKTOK_CLIENT_KEY`, `scope = TIKTOK_SCOPE`, `response_type = "code"`, `redirect_uri = process.env.NEXT_PUBLIC_APP_URL + "/api/auth/tiktok/callback"`, `state`.

Export async function `exchangeTikTokCode(code: string): Promise<{ accessToken: string; refreshToken: string; openId: string; expiresIn: number; refreshExpiresIn: number; scope: string }>` that POSTs to `TIKTOK_TOKEN_URL` with `Content-Type: application/x-www-form-urlencoded` body containing `client_key`, `client_secret = process.env.TIKTOK_CLIENT_SECRET`, `code`, `grant_type = "authorization_code"`, `redirect_uri`. Export async function `refreshTikTokToken(refreshToken: string): Promise<{ accessToken: string; refreshToken: string; expiresIn: number; refreshExpiresIn: number }>` that POSTs to `TIKTOK_TOKEN_URL` with `grant_type = "refresh_token"`, `client_key`, `client_secret`, `refresh_token`. Throws on error. Returns mapped fields.

All functions guard missing env vars (TIKTOK_CLIENT_KEY, TIKTOK_CLIENT_SECRET) by throwing `Error("TikTok OAuth env vars not configured")` at call time.
